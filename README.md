# ✨ AI Air Writing System V5

A real-time air drawing system using hand gestures powered by OpenCV and MediaPipe.

## 🚀 Features
- Draw in air using finger
- Gesture-based controls
- Multiple colors
- Eraser mode
- Save drawings

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy

## ▶️ Run the project
pip install -r requirements.txt  
python air_writing_v5.py
